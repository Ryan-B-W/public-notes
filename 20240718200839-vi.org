#+title: Vi
#+filetags: :vim:software:software_development:programming:computer_science:

An old visual text editor using modal editing.  Long popular as a rival to [[id:aca1324c-b142-4e34-a121-a8bb0a79ddf8][Emacs]].  Long since supplanted by [[id:37c53f5b-c586-41ff-a4fe-b44c05ed3c1f][Vim]].  Famed for the "h", "j", "k", and "l" navigation keys and so-called "Vi(m) motions".
