  * About Conkeror

  Conkeror is a keyboard-oriented, highly-customizable, highly-extensible web browser based on [[http://www.mozilla.org/][Mozilla]] [[http://developer.mozilla.org/en/docs/XULRunner][XULRunner]], written mainly in JavaScript, and inspired by exceptional software such as [[http://en.wikipedia.org/wiki/Emacs][Emacs]] and [[http://en.wikipedia.org/wiki/Vi][vi]].  Conkeror features a sophisticated keyboard system, allowing users to run commands and interact with content in powerful and novel ways.  It is self-documenting, featuring a powerful interactive help system.
