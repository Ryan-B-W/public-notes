  * Clang-Format Style Options

  [[https://clang.llvm.org/docs/ClangFormatStyleOptions.html][Clang-Format Style Options]] describes configurable formatting style options supported by [[https://clang.llvm.org/docs/LibFormat.html][LibFormat]] and [[https://clang.llvm.org/docs/ClangFormat.html][ClangFormat]].

  When using *clang-format* command line utility or =clang::format::reformat(...)= functions from code, one can either use one of the predefined styles (LLVM, Google, Chromium, Mozilla, WebKit, Microsoft) or create a custom style by configuring specific style options.
