#+title: (github.com) emacsorphanage/god-mode: Minor mode for God-like command entering
#+filetags: :lisp:emacs:software:website:

#+begin_quote
  This is a global minor mode for entering [[id:aca1324c-b142-4e34-a121-a8bb0a79ddf8][Emacs]] commands without modifier keys.  It's similar to [[id:37c53f5b-c586-41ff-a4fe-b44c05ed3c1f][Vim]]'s separation of command mode and insert mode.
  All existing key bindings will work in God mode.  It's only there to reduce your usage of modifier keys.
#+end_quote
