#+title: Emacs Lisp

[[id:aca1324c-b142-4e34-a121-a8bb0a79ddf8][Emacs]] [[id:84ae6e85-a6a2-4133-bc53-274238081c2d][Lisp]] is a dialect of the Lisp family of [[id:b24601aa-09df-41e1-aa7e-25ead342db34][programming languages]] used in the Emacs text editor.  A Lisp-1—a Lisp with only one symbol namespace for variables and functions—with features in common more with [[id:b5fb5c4b-d10f-4bca-91e1-a5e946ef0c83][Common Lisp]] than [[id:6246f8d4-6cd4-489d-b19f-9c1142b51b60][Scheme]] despite being a Lisp-1 like Scheme rather than a Lisp-2 like Common Lisp.
